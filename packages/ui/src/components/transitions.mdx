# Transitions

Orbit UI includes a reusable `t-*` motion layer from `transitions-dev`. These
classes are framework-agnostic CSS hooks: production components can opt in by
adding the documented classes and state attributes while keeping state ownership
in React.

The transition variables and snippets are imported through `@orbit/ui/styles.css`
via `src/transitions.css`.

## Available Hooks

- `t-digit-group` / `t-digit` for number pop-in counters.
- `t-badge` / `t-badge-dot` for notification badges on icon buttons.
- `t-icon-swap` / `t-icon` for swapping two icons in one slot.
- `t-skel` / `t-skel-skeleton` / `t-skel-content` for skeleton reveal.
- `t-tabs` / `t-tabs-pill` / `t-tab` for sliding segmented controls.
- `t-tt-wrap` / `t-tt-trigger` / `t-tt` for pure CSS tooltips.

Every snippet includes a `prefers-reduced-motion` guard in `transitions.css`.
