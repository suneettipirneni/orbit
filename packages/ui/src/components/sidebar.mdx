# Sidebar

Use the sidebar primitives for Orbit app navigation surfaces: deck libraries,
browser filters, and route-level navigation. The component is stateful through
`SidebarProvider`, supports collapsed icon mode, and keeps tooltip content
available for collapsed menu buttons.

## Deck Library

Full app-like composition with header identity, search, active menu items,
badges, submenu rows, footer actions, and rail collapse.
